# Captcha site

This website contains a little fun program to send a valentines message.
Just use 
```
love.d-solve.de?to=Name&from=Sender
```
to send a message to your love.

So use url parameters to customize the names on the website:
?to=Name&from=Sender


example
```
love.d-solve.de?to=Fredi&from=Felix
```
